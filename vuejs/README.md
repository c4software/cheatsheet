# VueJS 2.2

[Débuter en Français](https://fr.vuejs.org/v2/guide/index.html)


## VueCLI

Installation

```
$ npm install -g vue-cli
```

Créer un projet «application » depuis le template officiel
```
$ vue init webpack my-project
```

## Cycle de vie

* beforeCreate
* created
* beforeMount
* mounted
* beforeUpdate
* updated
* beforeDestroy
* detroyed
